Open Context Test Host

This development server hosts the new version of Open Context. This new version is still under active development and this server will occasionally go off-line as new updates get installed.

We welcome help and identification of any problems. Please provide feedback via Open Context's GitHub repository.

We gratefully acknowledge and thank the German Archaeological Institute (DAI) for hosting and mirroring this new version of Open Context.

Open Context is a publishing service maintained by the Alexandria Archive Institute, a 501(c)3 nonprofit organization. Open Context data is archived by the California Digital Library.
