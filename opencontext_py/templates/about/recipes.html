Data Analysis with Open Context

Open Context has a powerful Application Program Interface (API) that enables you to independently:

  • Search and browse data
  • Visualize data
  • Analyze data
  • Link to Open Context records (entity reconciliation)
  • Interoperate with Open Context

API Overview

Open Context has two primary APIs. One returns data on individual records, and the other returns data from searches.
